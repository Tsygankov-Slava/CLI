Usage:
   command [flags] [arguments]

Command:
  printFlagsAndArguments  Displays 
                          the passed
                          flags and
                          arguments
	Flags:
      -d, --dir=VALUE     A flag  
                          that acc-
                          epts a 
                          directory
                          as input.
	Example:
      $ printFlagsAndArguments file1.txt --dir value file2.txt
      >>> Flags:
          flag -> value
       Arguments:
          file1.txt
          file2.txt
